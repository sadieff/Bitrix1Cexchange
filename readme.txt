Условия от  клиента: из 1С приходят товары с торговыми предложениями. Торговые предложения приходят в одном файле с товарами - import.xml
В 1С заведено много складов, но использовать необходимо редакцию Битрикс "Малый бизнесс" с 1 складом. 

Задача: создать структуру каталогов из товаров: Бренд->Модель->Товар. Товары на сайте простые, без торговых предложений. 

Файл 1c.php - создание структуры каталогов и размещение товара в нужном разделе или обновление свойств у уже существующего товара.

Файл 1с-store-price.php - получение товара из файла импорта, поиск его торгового предложения и добавление\обновление цены у товара в каталоге. Получение данных о количестве со складов и сохранение этих данных в специально заведенные свойства товара.
